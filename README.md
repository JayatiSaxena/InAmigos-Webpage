# InAmigos-Webpage
# InAmigos Foundation NGO Awareness Webpage

## Overview

This project was developed as part of an internship assignment to create an awareness webpage for InAmigos Foundation. The objective was to design and develop a responsive and informative website that highlights the NGO's mission, ongoing projects, social impact, and opportunities for community participation.

## Features

* Introduction to InAmigos Foundation and its vision.
* Dedicated sections showcasing ongoing projects and initiatives.
* Social impact statistics to demonstrate the organization's contributions.
* Responsive and user-friendly design using HTML and CSS.
* Interactive navigation bar with NGO branding and logo.
* Call-to-action section encouraging users to volunteer and support the cause.
* Contact and social media section for engagement and outreach.

## Technologies Used

* HTML5
* CSS3
* Google Fonts (Poppins)

## Learning Outcomes

Through this assignment, I gained hands-on experience in:

* Structuring webpages using semantic HTML.
* Creating responsive layouts with CSS.
* Designing visually appealing user interfaces.
* Organizing content effectively for awareness campaigns.
* Building real-world projects based on client requirements.

## Purpose

The webpage aims to spread awareness about the impactful work carried out by InAmigos Foundation and encourage individuals to contribute towards social change through volunteering, participation, and support.
